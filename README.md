# PerfectSky Post Bot
Bot Account  
https://bsky.app/profile/perfectskypost.bsky.social

The PerfectSky Post Bot analyzes the Trending Feed and publishes:

- Character and word averages  
- Majority media type  
- Dominant post type (reply or quote when relevant)  

Runs daily through GitHub Actions with no manual intervention.
